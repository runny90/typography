Abbreviations

It is used to mark up abbreviation or acronym:

cdglwas founded in 2020.

Blockquote

It is used to present content from another source:

Codingal is on a mission to inspire all the school kids

Multiple code lines

For multiple line of code, use the pre element:

    codingal is on the mission to inspire all the school kids

Keyboard Inputs

To indicate input that is typically entered via the keyboard, use the kbd element
